# politutorias-prompts

Compendio público de los prompts de IA Generativa (modelo **Gemini 2.5 Flash**) utilizados durante el desarrollo del componente **DevOps y orquestación de IA agéntica** del proyecto **Politutorías**, Trabajo de Integración Curricular de la Escuela Politécnica Nacional.

**Total de prompts publicados:** 253 en 8 secciones temáticas.

## Secciones

1. [Migración a AWS](01-migracion-aws/) — 64 prompts
2. [PoC n8n local](02-poc-n8n-local/) — 2 prompts
3. [Agente supervisor](03-agente-supervisor/) — 3 prompts
4. [Orquestación n8n productiva](04-orquestacion-productiva/) — 103 prompts
5. [FEDORA Local](05-fedora-local/) — 18 prompts
6. [Vercel para el frontend](06-vercel/) — 34 prompts
7. [Redacción TIC DevOps](07-tesis-devops/) — 24 prompts
8. [Ensayo DevOps](08-ensayo-devops/) — 5 prompts
